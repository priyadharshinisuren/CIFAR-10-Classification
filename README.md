# CIFAR-10-Classification
## 📌 Overview This project explores the use of **Recurrent Neural Networks (RNNs, LSTMs, GRUs)** for image classification on the **CIFAR-10 dataset**.

- Dataset: CIFAR-10 (60,000 images, 10 classes)
- Architectures: RNN, LSTM
- Goal: Compare performance of sequence models vs CNNs
